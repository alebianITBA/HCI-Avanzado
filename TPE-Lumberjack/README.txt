El juego ejecutable se encuentra tanto para Windows como para Mac
dentro de la carpeta "Juego". Para salir del juego en Windows,
se debe presionar la tecla de Inicio y cerrar la ventana minimizada.

La carpeta "Código" incluye todos los scripts de C# utilizados
en el Proyecto de Unity 3D.

La carpeta "Proyecto Unity 3D" contiene el proyecto en su totalidad.
Se requiere tener instalado Unity 3D versión 5.0.2.

La carpeta  "Informe" incluye el informe en formato PDF.
